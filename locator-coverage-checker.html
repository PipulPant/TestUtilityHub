Best Practice Tips
  • Prefer data-testid attributes — they're stable and automation-specific.
  • ID selectors like #submit-btn are unique by definition — great for automation.
  • Avoid deeply nested selectors like .a > .b > .c > span — they break on layout changes.
  • Never use :nth-child() — the index shifts when elements are added or removed.
  • Dynamic class names like .sc-abc123 (CSS-in-JS) change on every build.
  • In Playwright, prefer getByTestId(), getByRole() or getByLabel() over raw CSS selectors.
